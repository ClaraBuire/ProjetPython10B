Pour lancer l'interface, lancer le fichier main.py
ATTENTION le cryptage image (2e onglet) ne fonctionne que sous linux à cause des chemins d'accès

Pour executer les modules sans l'interface :
- cryptage texte dans une image et image dans une image, executer codage_image.py
- pour les cryptages César et Vigenere, appeller les fonctions cesar ou vigenere du module cryptagetexte.py
  Ces fonctions prennent en entrée une une clé (module cle.py) et un booléen (True pour cryptage, False pour décryptage)
